# Kaggle_project-for-5559
I worked on the tabular data and tested several approaches. CatBoost performed the best, reaching around 1.61 RMSE with a single model. I also tried text features and XGBoost, but they performed worse and didn’t help. Then I implemented a 5-fold K-Fold CatBoost, which gave similar performance (~1.60) but more stable results. I also tried simple ensemble methods, but only combining strong models (single CatBoost + K-Fold) is potentially helpful. At this point, the performance seems to have plateaued, so further improvements will likely come from better ensemble strategies or feature engineering rather than new models.
